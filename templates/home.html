Welcome to ASCENDS

ASCENDS helps researchers run practical machine learning on local CSV data with a guided workflow.

What ASCENDS is for
  • Understand feature-target relationships quickly
  • Train baseline regression or classification models without coding from scratch
  • Save runs and generate predictions on new data
Recommended Workflow
  1. Correlation Analysis & Feature Selection: Upload CSV, choose target, and identify useful inputs.
  2. ML Performance Training & Test: Train a model, review metrics/plots, and save the run.
  3. ML Prediction: Apply the saved model to new CSV data.
